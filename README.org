#+title: Common Lisp interface to Isocline with a REPL

[[https://github.com/daanx/isocline][Isocline]] is an alternative to libreadline, libedit and the likes, that is

- MIT licensed
- Pure C
- Portable across Unix, Windows, MacOS
- Supports multiline editing out of the box
- ... and much more ...

This means it works as an awesome portable repl for Common Lisp, across operating systems and lisp implementations.

The current repository contains source code for version 1.0.9 of Isocline, along with [[file:lisp/isocline.lisp][lisp bindings]] and a [[file:lisp/isocline-repl.lisp][simple repl]].

* Installation

** Binaries

Currently standalone binaries are available for Windows. See the [[https://github.com/digikar99/cl-isocline/releases/][Releases]].

** Compiling from source

*** Quicklisp client

[[https://github.com/rudolfochrist/ql-https][rudolfochrist/ql-https]] is a quicklisp client variant that uses https.

#+begin_src sh
curl https://raw.githubusercontent.com/rudolfochrist/ql-https/master/install.sh | bash
#+end_src

*** Ultralisp

#+begin_src lisp
;; ql-https will upgrade the URLs from http to https
(ql-dist:install-dist "http://dist.ultralisp.org/" :prompt nil)
#+end_src

*** isocline-repl

If you have ultralisp installed, then you can

#+begin_src lisp
(ql:quickload "isocline-repl")
(isocline-repl:run)
#+end_src

*** Binary

The following should create a =cl-isocline-repl= in the root directory of =isocline-repl=.

#+begin_src lisp
(asdf:make "isocline-repl")
#+end_src
